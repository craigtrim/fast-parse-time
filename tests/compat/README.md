# Compatibility Tests

These tests verify compatibility with leading NLP date and time parsing libraries:

- [datefinder](datefinder/) - compatibility tests for the `datefinder` library
- [dateparser](dateparser/) - compatibility tests for the `dateparser` library
- [parsedatetime](parsedatetime/) - compatibility tests for the `parsedatetime` library

Each subdirectory contains tests that validate `fast-parse-time` produces equivalent or improved results compared to the corresponding library for common date and time parsing scenarios.
